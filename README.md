# 🚀 Getting Started

1. Clone the repository.
2. Open the project in your preferred IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Configure your MySQL database in `application.properties`.
4. Run the Spring Boot application.
5. Test the APIs using Postman or connect them with the frontend.
